# Style Tokens

## Colors

| Token | Value | usage |
|---|---|---|
| `primary` | `#f6c71e` | Buttons, Highlights, Icons |
| `text-main` | `#414042` | Main text (light mode) |
| `background-light` | `#ffffff` | Body background (light mode) |
| `background-dark` | `#0f171a` | Body background (dark mode) |
| `deep-teal` | `#19272b` | Dark sections, Footer |
| `deep-teal-dark` | `#0d1417` | Dark mode alternative |
| `divider` | `#d9d9d9` | Borders |

## Fonts

| Token | Family |
|---|---|
| `font-display` | Sora |
| `font-body` | DM Sans |

## Radii

*   `rounded-xl`: 12px (Form inputs)
*   `rounded-2xl`: 16px (Buttons, standard cards) - *Default*
*   `rounded-3xl`: 24px (Hero image container)
*   `rounded-[32px]`: Cards
*   `rounded-[48px]`: Large images
